# 西梅时钟

## 简介

这款软件主打一个成分复杂！（哈哈哈）

起这个软件名字，是因为今天我种了一棵西梅树，还因为我喜欢的球星是梅西（哈哈哈）

## 软件功能

本来受软件Pomy启发，想做个遵循 20-20-20 视力保护原则的休息提醒工具（但因为pomy是商店的软件，我用的精简版系统懒得装商店，于是乎自己写个吧）

最近在重温《命运石之门》，既然是钟表，那就用辉光管吧（泰酷辣！）

切换功能——点击右键

* 功能二：遵循 20-20-20 视力保护原则的休息提醒工具
